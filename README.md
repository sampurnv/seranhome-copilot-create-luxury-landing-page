# Seran Home - Intelligent Living. Effortlessly Luxurious.

An ultra-luxury, high-end landing page for a premium home automation brand targeting villa owners, penthouse residents, and luxury real estate developers.

Built with **React**, **JavaScript**, **CSS**, and **HTML**.

## Overview

Seran Home showcases a sophisticated, minimal design that emphasizes lifestyle and experience over technical features. The landing page conveys exclusivity, craftsmanship, and modern luxury through elegant design and refined copy.

## Features

- **Hero Section**: Full-screen cinematic hero with bold headline and primary CTA
- **Value Proposition**: Highlighting bespoke design, invisible technology, and white-glove service
- **Lifestyle Features**: Five premium sections covering lighting, climate, security, entertainment, and curtains
- **Technology Partners**: Trusted brands including Crestron, Control4, KNX, Lutron, and Savant
- **4-Step Process**: Consultation → Design → Installation → Handover
- **Project Showcase**: Premium portfolio of exceptional homes
- **Testimonials**: Refined client testimonials with elegant typography
- **Contact Section**: Simple call-to-action for consultations
- **Footer**: Clean, understated layout with contact information and social links

## Design Philosophy

- **Color Palette**: Dark, sophisticated (charcoal black, deep grey) with gold/champagne accents
- **Typography**: Large, generous white space, cinematic presentation
- **Animations**: Smooth transitions, subtle parallax effects, fade-in on scroll
- **Responsiveness**: Fully responsive design for all screen sizes

## Technologies

- **React 18** - Component-based UI library
- **Vite** - Fast build tool and dev server
- **CSS3** - Custom properties, Grid, Flexbox
- **JavaScript (ES6+)** - Modern JavaScript with hooks and effects

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sampurnv/seranhome.git
cd seranhome
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

This will create an optimized production build in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```
